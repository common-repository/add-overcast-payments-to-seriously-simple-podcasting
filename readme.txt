If you wanted to add support for Overcast payments to your podcast RSS feed this is the plugin that will do it.

== Installation ==

Installation is simple, either install automatically from the WordPress plugins repository from admin interface of your website, or, upload via FTP.

1. Upload the `/add-overcast-payments-to-seriously-simple-podcasting/` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update the settings in the Seriously Simple Podcast settings page.
